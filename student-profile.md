# Student Profile

## 기본 정보
- **분류**: C-제조
- **산업**: 자동차 부품 제조
- **역할**: 제조 실무자 (분석 + 의사결정 권한 보유)

## 기술 역량
- **프로그래밍**: Python, SQL 능숙
- **데이터 도구**: Excel 능숙
- **수준**: 중상급 (도구 활용에 어려움 없음)

## 데이터 환경
- **MES**: CSV 추출
- **DB**: 직접 조회
- **ERP**: 시스템 연동
- 데이터 접근성 양호, 다양한 경로 확보

## 학습 목표
- **핵심 목표**: 공정 데이터 분석을 통한 불량률 감소
- **현재 고민**: 데이터는 충분하나 분석 방법론(어떻게 분석할지)이 부족
- **배우고 싶은 것**: 시각화 대시보드 구축

## 결과 활용
- 상위 관리자 보고용 자료 작성
- 본인 직접 의사결정에 활용

## 추천 학습 방향
1. **공정 데이터 분석 방법론**: SPC, 파레토 분석, 상관분석 등 제조 특화 분석 기법
2. **시각화 대시보드**: Streamlit / Plotly / Power BI 등을 활용한 실시간 모니터링
3. **불량 원인 탐색**: 다변량 분석, 의사결정 트리 등 원인-결과 규명 기법
4. **보고서 자동화**: 분석 결과를 경영진 보고용으로 자동 생성하는 파이프라인
