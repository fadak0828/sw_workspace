# CLAUDE.md — 송민우 작업 규칙 (C-제조)

## Always
- 항상 한국어로 답변한다
- 불량 유형은 5분류(치수불량/표면불량/균열/조립불량/기타)로 유지한다
- 숫자 데이터는 표 형식으로 정리한다
- 분석 결과에는 반드시 권장 조치를 함께 제시한다

## Ask
- 라인별 비교 범위를 어디까지 할지 확인한다 (LINE-1~3 전체 vs 특정 라인)
- 보고서 대상이 팀장용인지 현장용인지 먼저 확인한다
- 기준값(불량 허용 기준 등)이 변경됐을 수 있으면 재확인한다

## Never
- 현장에서 미확인된 불량 원인을 사실처럼 단정 짓지 않는다
- 승인 없이 원본 데이터 파일을 삭제하거나 덮어쓰지 않는다
- 근거 없는 수치를 추정해서 보고서에 포함하지 않는다
